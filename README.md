Install Go: https://go.dev/doc/install
Install gopls: go install golang.org/x/tools/gopls@latest
Install tmux, neovim.
Install stow.

After downloading the repo, `git clone "https://github.com/Rando-Khan/dotfiles.git"`,
enter the dotfiles directory, and run `stow . --adopt`.

run :MasonInstallAll in nvim.
